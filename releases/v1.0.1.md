# 小白 AI 工具箱 v1.0.1

本次发布修复 Windows 安装包和 codex汉化增强版联动相关问题，继续提供预编译安装包。用户不需要 Go、Node 或 NSIS 等构建环境。

## Highlights

- 修复安装目录、版本号和执行路径的注册表写入，确保 codex汉化增强版可以稳定找到小白 AI 工具箱。
- 完善安装包 manifest 信息，便于本地桥接识别工具箱版本和启动入口。
- 支持静默安装，方便随 codex汉化增强版自动安装或更新。

## Notes

- GitHub Release 中面向用户的是预编译 Windows 安装包，不要求用户下载源码自行构建。
- 登录、付款、实名、充值和云资源操作仍保留给用户自己确认。
