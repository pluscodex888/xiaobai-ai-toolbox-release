# 小白 AI 工具箱 v1.3.0

本次正式发布重点改进多账号切换稳定性、版本辨识和小白社区分类卡片展示。用户不需要 Go、Node 或 NSIS 等构建环境。

## Highlights

- 多账号页面在本地接口短暂重启或探测失败时保持当前页面，不再自动跳回“热门辅助工具”。
- VS Code Codex 同步提示不再展示具体的全局登录文件名，只说明同步和自动备份结果。
- 顶部“小白AI工具箱”标题后以浅灰色显示当前正式版本号。
- 小白社区回答中的 AI 服务商分类信息块同步问答站卡片样式，不再退化为普通文本。
- 分类卡片保留“查看分类”入口，并继续通过工具箱的社区打开流程访问。

## Notes

- GitHub Release 中面向用户的是预编译 Windows 安装包，不要求用户下载源码自行构建。
- 登录、付款、实名、充值和云资源操作仍保留给用户自己确认。
- 本次版本保持旧配置与既有热更新流程兼容。
