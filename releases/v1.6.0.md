# 小白AI工具箱 v1.6.0

本次正式发布面向 Windows 用户，发布资产为已经编译好的安装包、校验文件和版本 manifest。

## 更新内容

- 社区答案现在支持安全渲染图片，图片会限制在内容区域内并使用懒加载。
- 优化社区答案里的代码块显示，避免代码区域被异常拉高或撑破页面。
- 启动或重启 Codex汉化增强版时减少批处理、PowerShell 辅助窗口闪烁。
- 读取 Windows 快捷方式和注册表信息时隐藏临时命令窗口，减少桌面打扰。

## 发布资产

- `xiaobai-ai-toolbox-setup-v1.6.0.exe`
- `xiaobai-ai-toolbox-setup-v1.6.0.exe.sha256`
- `xiaobai-ai-toolbox-setup-v1.6.0.json`

GitHub Release 中面向用户的是预编译 Windows 安装包，不要求用户下载源码自行构建。
