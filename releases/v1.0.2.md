# 小白 AI 工具箱 v1.0.2

本次正式发布修复默认问答站域名和 Codex 联动场景下的链接生成问题，继续提供预编译 Windows 安装包。用户不需要 Go、Node 或 NSIS 等构建环境。

## Highlights

- 默认问答站地址切换到 www 主域名，旧本地配置会自动迁移。
- 修复问答接口返回旧域名时，工具箱内展示和跳转仍指向旧地址的问题。
- 保持 Codex 侧调用工具箱、国产模型引导和社区问答入口的兼容性。
- 重新编译并验证 Windows 安装包，包内不包含 Go/TS 源码或现场构建脚本。

## Notes

- GitHub Release 中面向用户的是预编译 Windows 安装包，不要求用户下载源码自行构建。
- 登录、付款、实名、充值和云资源操作仍保留给用户自己确认。
