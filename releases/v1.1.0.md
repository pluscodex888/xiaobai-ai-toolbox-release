# 小白 AI 工具箱 v1.1.0

本次正式发布新增 Codex 多账号切换对 VS Code Codex 插件的可选同步能力，并继续提供预编译 Windows 安装包。用户不需要 Go、Node 或 NSIS 等构建环境。

## Highlights

- 多账号切换页新增“同步到 VS Code Codex 插件”勾选开关，默认关闭。
- 勾选后切换 Codex 原生账号时，会同步目标账号登录态到全局 `~/.codex/auth.json`。
- 写入 VS Code Codex 登录态前会自动备份原 `auth.json` 到 `~/.codex/backups/`。
- 切换确认弹窗会明确提示是否影响 VS Code Codex 插件。
- 保持桌面 codex汉化增强版 与 VS Code Codex 插件默认隔离，不勾选时不影响 VS Code。

## Notes

- GitHub Release 中面向用户的是预编译 Windows 安装包，不要求用户下载源码自行构建。
- 登录、付款、实名、充值和云资源操作仍保留给用户自己确认。
