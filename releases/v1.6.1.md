# 小白AI工具箱 v1.6.1

本次正式发布面向 Windows 用户，发布资产为已经编译好的安装包、校验文件和版本 manifest。

## 更新内容

- 修复模型“充值”流程：打开模型平台官网后会在登录完成时跳转到充值页，并保持浏览器窗口，不再自动关闭。
- 充值和支付过程中不再显示右侧浮动引导面板，避免面板在支付页反复闪烁或遮挡操作。
- 安装和更新时用于结束旧进程的辅助命令改为隐藏执行，避免用户交互过程中出现控制台黑框闪烁。
- 修复从不同工作目录或不同入口重复启动时可能出现两个小白工具箱主界面的问题，工具箱界面继续保持单例。
- 充值模式不再触发 API Key 创建、余额自动保存或完成后关窗逻辑，只负责把用户带到充值位置。

## 验证记录

- 安装包版本：`1.6.1`
- 安装包：`xiaobai-ai-toolbox-setup-v1.6.1.exe`
- 运行路径：`C:\Users\Administrator\AppData\Local\XiaoBaiAIToolbox\小白AI工具箱.exe`
- 运行验证：真实安装后启动应用，首页返回 200，界面显示 `v1.6.1`。
- 关键入口：模型充值入口、模型状态接口、Codex汉化增强版账号/额度入口均可访问。
- 配置检查：`%APPDATA%\codex-plusplus\desktop-codex-home\config.toml` 未发现重复 key，`[features].plugins`、`sandbox_mode`、`approval_policy`、`[windows].sandbox` 和增强版 MCP env 表均已检查。
- 安装器检查：生成的 NSIS 脚本不再包含 `ExecWait taskkill`，改为隐藏执行。
- 单例检查：重复启动工具箱时只保留一个主界面，旧的工具箱浏览器壳会被复用、聚焦或清理。
