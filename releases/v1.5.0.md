# 小白AI工具箱 v1.5.0

本次正式发布面向 Windows 用户，发布资产为已经编译好的安装包、校验文件和版本 manifest。

## 更新内容

- 服务器信息支持保存多台服务器，新增服务器不会覆盖已有服务器记录。
- 服务器列表改为每台服务器独立节点，每个节点拥有自己的映射域名、AI部署、购买/更换、续费或删除操作。
- 自录入服务器卡片隐藏登录账号和密码掩码，只保留必要服务器信息。
- 优化服务器信息卡片层级，用更清晰的深浅底色区分外层面板和服务器节点。
- 添加服务器弹窗不再因为鼠标移出表单区域自动关闭，只能通过取消或保存关闭。
- Codex汉化增强版读取部署服务器时支持多服务器列表，并会按用户选中的服务器 IP 获取对应部署上下文。
