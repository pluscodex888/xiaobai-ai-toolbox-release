# 小白 AI 工具箱 v1.2.0

本次正式发布增强热门辅助工具对小白社区问答内容的兼容性，并改进软件下载前的登录提示流程。用户不需要 Go、Node 或 NSIS 等构建环境。

## Highlights

- 热门辅助工具现在可以正确渲染小白社区回答里的软件下载卡片。
- 兼容 `software-info-card` 卡片样式，不再把卡片 HTML 当作普通文本显示。
- 软件下载卡片会使用小白社区登录态创建下载 ticket，再打开真实下载地址。
- 未登录点击下载时，只弹出登录/注册窗口，不再提前显示右下角“正在打开小白社区页面”提示。
- 同步重建命令行程序和两个 Windows GUI 入口，避免入口版本不一致。

## Notes

- GitHub Release 中面向用户的是预编译 Windows 安装包，不要求用户下载源码自行构建。
- 下载软件前仍需要登录小白社区账号。
- 登录、付款、实名、充值和云资源操作仍保留给用户自己确认。
